# D'coratto Document Editor

Aplicacao para criar documentos estruturados de projeto inicial e exportar PDFs premium.

O frontend oficial atual e o HTML legado copiado para `public/editor_projeto_inicial.html`. A aplicacao React carrega esse HTML como tela principal enquanto a persistencia Supabase e o preview final sao estruturados ao redor dele.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Deploy

Depois do build, publique a pasta `dist` em Vercel, Netlify, GitHub Pages ou outro host estatico.

## Arquitetura

Leia `docs/ARQUITETURA.md` antes de alterar o fluxo principal.
